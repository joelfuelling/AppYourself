About AppYourself

App Yourself is a Userbased app to add/edit/delete roles applied to or interested in!

Add a record for each followup on each role (name, contact info., date stamp) and be alerted via text of the number of days elapsed since your last follow up.

An extra visual reminder will display if there's no followup added within 5 days of the last followup.

Create, then add or remove tags to roles to categorize industry, type of work, languages used, etc.

Don't forget to enter a link to the role or website when creating the role. That link can be clicked on the individiual roles page.
